About

This is a lightweight, static web UI concept for image upscaling workflows inspired by the NCNN ecosystem. The goal is to keep the experience privacy-friendly (client-side when possible), simple to run, and easy to extend.

What you can do here

  • Browse algorithm presets and sample results.
  • Prepare an upscaling job (image + settings) in a clean UI.
  • Extend the project with JavaScript/TypeScript modules.

Open-source credits

  • NCNN
  • Real-ESRGAN
  • Real-CUGAN
  • waifu2x

Notes

If you plan to run real NCNN models in the browser (WebAssembly/WebGPU), add the runtime as a separate module and keep model files out of the repository unless you have redistribution permission.
